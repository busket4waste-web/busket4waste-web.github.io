# Прототип: цель для чаевых (курьерское приложение)

Интерактивный UI-прототип фичи «Моя цель» в экране чаевых. Референс дизайна — скриншоты профиля в `screenshots/`.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
# Например, через Python:
python -m http.server 8080
# или
npx serve .
```

После этого откройте в браузере `http://localhost:8080` (или путь к `index.html`).

## Что внутри

1. **Экран «Чаевые»**
   - Блок «Моя цель»: при пустой цели — CTA «Добавить цель», при заполненной — текст цели и кнопка «Изменить».
   - Переключатель «Показывать цель покупателям».
   - Мелкий текст про приватность.

2. **Экран «Редактирование цели»**
   - Заголовок «На что ты копишь?»
   - Пресеты (чипы): Телефон, Ремонт, Отпуск, Обучение, Подарок, Долги / кредиты, Другое.
   - Текстовое поле до 80 символов, placeholder «Например: на новый телефон».
   - Подсказка: «Не указывай личные данные или контакты».
   - Кнопки «Сохранить» и «Отмена».

Переход на редактирование — по «Добавить цель» или «Изменить». Возврат — по стрелке назад или «Отмена». Состояние цели и переключателя сохраняется в рамках сессии (в памяти).

## Цель прототипа

Собрать UX/UI-фидбек по понятности, комфорту и готовности курьеров заполнять цель. Деньги и бэкенд не задействованы.

## Тестирование у курьеров

Чтобы разослать прототип курьерам и собирать ответы (цель + тогглы «Показывать цель» и «Показывать фото»), внизу экрана «Чаевые» есть кнопка **«Отправить»**. Подробная инструкция: как включить приём ответов (Formspree или копирование), куда выложить прототип и что написать курьерам — в файле [TESTING.md](TESTING.md).
